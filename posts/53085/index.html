WordPress著名SEO插件Yoast SEO出现重大BUG，将导致网站排名下降

发表于 2018-06-05 | 分类于 SEO |
本文字数： 1.8k | 阅读时长 ≈ 3 分钟

Yoast SEO近期更新了一个版本，修复了7.0版本中会导致一些网站排名下降的一个重大BUG。Yoast的创始人Joost de Valk向收到该BUG影响，导致网站失去排名的站长们发表了个人道歉声明。

Yoast SEO 7.0 Bug上线于2018年3月6号

Yoast SEO插件出现BUG的7.0版本上线于2018年3月6号。使用Yoast SEO插件，并且在3月份升级到了7.0 BUG版本的网站，如果出现网站排名下降，那很有可能是受到了该BUG的影响。

Yoast SEO插件到底出了什么BUG呢？

原来，Yoast SEO插件升级到7.0版本后，有一项升级改动影响到了图像重定向设置方式。Yoast官方称：所幸这次升级并没有对所有用户造成影响。

Yoast官方对此次问题的声明为：

这是一个低级但是愚蠢的BUG，当Yoast SEO从低版本升级到7.0-7.0.2版后，并没正确覆盖掉旧版本的插件设置。并且由于这次更新是重写了旧版的设定，导致后续也无法对BUG进行回滚覆盖。

Yoast创始人也对此发表了个人道歉声明

声明中表示他们很快就发现了该7.0版的BUG，并且快速的发布了修改版。但是自3月6日以来，该错误已经对部分网站的SEO产生了负面影响。因此，Yoast的创始人Joost De Valk个人道歉如下:

“This post serves both as a warning and an apology. …We’re so very sorry. …we messed up. I myself, am sorry. More so than normal, because I came up with and coded this change myself…”

受此影响的Yoast SEO用户也抱怨连连

一位插件使用者发文表示，由于Yoast SEO插件的BUG，他的网站搜索排名下降。由于这个错误他已经对Yoast SEO插件永远失去了信心。

还有一位插件使用者在Yoast的公告下面发表评论，表示由于Yoast SEO插件的问题，导致她的网站排名下降，并因此损失了数千美元。不过她同时表示，在更新了修复版本后，她的网站排名已经恢复了。

哪些网站会受到此次Yoast SEO插件BUG的影响呢？

据Yoast官方声明称，并不是所有用户都会受到影响。但是如果你的网站排名在3月6号之后下降了，最好查看下Yoast SEO插件的媒体设置选项。

如何查看Yoast SEO插件的媒体设置

  1. 打开WordPress侧边栏导航；
  2. 选择Yoast SEO插件设置；
  3. 点击搜索外观选项（Search Appearance）。
  4. 在搜索外观设置选项卡中点击媒体（Media）

然后查看媒体设置中“将附件 URL 重定向到附件本身？”选项。根据实际情况有以下4种处理方式：

情况1：一切正常

如果媒体设置中“将附件 URL 重定向到附件本身？”选项设置为“是”。那么说明一切正常，你只需要把Yoast SEO插件更新到最新版本就可以了。

情况2：网站排名未受影响

如果媒体设置中“将附件 URL 重定向到附件本身？”选项设置为“否”，但是网站排名并未受到影响。那只需要把该设置改为“是”，并把插件更新到最新版本即可。

情况3：网站排名下降

如果媒体设置中“将附件 URL 重定向到附件本身？”选项设置为“否”，并且自3月6号后网站排名下降，那就说明网站SEO受到了7.0版bug的影响。

Yoast官方提供的解决办法是将媒体设置中“将附件 URL 重定向到附件本身？”选项设置为“是”，并将插件更新到最新版本。此外，还需要下载Yoast SEO搜索索引清除插件并应用此插件以修复该错误。

情况4：如果你还不确定是否受到BUG影响

Yoast官方建议将此设置改为“是”，并将插件更新到最新版本。
